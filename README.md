# Expense Tracker

A command-line expense tracker built in Python to manage daily expenses using JSON file storage.

## 🚀 Features
- Add new expenses  
- View all expenses  
- Calculate total expenses  
- Data stored using JSON file  

## 🛠️ Tech Used
- Python  
- JSON (File Handling)

## ▶️ How to Run
1. Clone the repository
2. Open the project folder
3. Run the file:
   python main.py

## 📂 Project Structure
- main.py
- expenses.json (auto-generated)
